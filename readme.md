## 📋 Sobre o Projeto

Este projeto consiste em uma aplicação web completa que utiliza a API do Google Gemini para gerar redações personalizadas seguindo todos os critérios de correção do ENEM. O sistema conta com uma interface moderna em Tailwind CSS e um backend robusto em Flask.

### 🎯 Funcionalidades Principais

- ✏️ **Temas Personalizados**: O aluno digita qualquer tema e a IA gera uma redação única
- 📝 **Estrutura Completa**: Título, introdução, desenvolvimento (2-3 parágrafos), conclusão e referências
- ⭐ **Análise das 5 Competências ENEM**: Avaliação detalhada baseada nos critérios oficiais
- 💡 **Temas Sugeridos**: Banco com 10+ temas prontos para prática
- 📋 **Cópia Rápida**: Botão para copiar a redação completa
- 🎨 **Design Moderno**: Interface responsiva com Tailwind CSS e animações suaves
- ⚡ **Atalho de Teclado**: Ctrl+Enter para gerar redação rapidamente

## 🚀 Tecnologias Utilizadas

### Backend
- **Python 3.8+**
- **Flask** - Framework web
- **Google Gemini API** - IA para geração de redações
- **Flask-CORS** - Suporte a requisições cross-origin

### Frontend
- **HTML5**
- **Tailwind CSS** - Framework CSS para design moderno
- **JavaScript** - Interatividade e consumo da API
- **Font Awesome / Emojis** - Ícones e elementos visuais
